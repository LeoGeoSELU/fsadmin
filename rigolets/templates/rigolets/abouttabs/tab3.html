Overview

The client is developing an algorithm that will take data from various sources and process it to determine optimal fishing locations. He wants a website that can handle the data processing on the backend and display the results on the frontend for end users.


Gather Data

  • Pull data from NOAA and USGS buoys.
  • Weather Conditions.
  • Analyze area for geographic dimensions.

Process and store

  • Implement client's algorithm.
  • Store results in database.
  • Provide ability for client to analyze and tweak results

Display

  • Display data on website using mapping libraries.
  • Provide API access to data for future developments.
